# TELEGRAM BOT - INSTALLATION GUIDE

##  Инструкция по установке на RDP сервер:

### 1. Установка Python:
   - Скачать Python 3.9+ с https://python.org
   - При установке ОБЯЗАТЕЛЬНО поставить галочку "Add Python to PATH"

### 2. Установка зависимостей:
   `
   pip install -r requirements.txt
   `

### 3. Запуск бота:
   - Запустить файл START_BOT.bat
   - Или в командной строке: python run_telegram_bot.py

### 4. Автозапуск при загрузке Windows:
   - Добавить START_BOT.bat в автозагрузку
   - Или настроить через Планировщик задач

### 5. Проверка работы:
   - Открыть Telegram
   - Найти бота @info_xm_trust_bot
   - Нажать /start
   - Проверить генерацию сигналов
